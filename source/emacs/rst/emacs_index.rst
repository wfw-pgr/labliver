=========================================================
古き良き高機能 editor :: *Emacs*
=========================================================

* PC普及前の黎明期 ( 1970- ) からある *GNU* 発の **Unix系の代表的 editor**
* **拡張性が高い** ことが特徴 ( *Emacs* を書くための専用言語 : **Emacs-Lisp** を使って自分なりにアレンジできる )
* エディタ戦争やってた二大巨塔の片割れ．なお現在は、 **レガシー** とか **枯れた** とか **オワコン** とかいう評判 ( オワコン扱いされてから長い )．
