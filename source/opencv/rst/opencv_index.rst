=========================================================
openCV とは？
=========================================================

.. note::

   OpenCV（オープンシーヴィ、英: Open Source Computer Vision Library）とはインテルが開発・公開したオープンソースのコンピュータビジョン向けライブラリ ( 引用：wikipedia-OpenCV )

          
* 主に、**画像処理** に使用される．最近では機械学習を用いた画像認識の前後処理などに用いられることが多い．
* 環境は、プログラミング言語としては、C/C++、Java、Python、MATLABなどにライブラリが開発されている．プラットフォームは、macOS, Unix系OS、Windows、Android、iOS等をサポートしている．
* 本稿では、主にpythonから使用する．
