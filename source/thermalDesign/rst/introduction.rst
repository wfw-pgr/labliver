=========================================================
熱設計の意義・重要性
=========================================================

* 旧来の熱設計は，熱による誤動作や経年劣化は，問題が顕在化した時のみに考える，場当たり的，対処療法的，付加的な技術であった．( つまり，本質的には， :red:`熱設計` ではなく，  :red:`熱対策` である )．
* 2000年代以降，デバイスの集積化，製品の **小型化** が進み，製品設計当初から熱を考慮しないと製品が実現しないようになってきている．熱設計は，実用的な製品開発を進める上で， **不可欠** である．


=========================================================
熱設計によって受けられる恩恵
=========================================================

* **故障・誤動作・事故の防止**
* **部品の経年劣化の阻止** ( 電解コンデンサ：使用温度によって蒸発量増加 )
* 機器温度上昇による不快感の低減
* 消費電力低減
* 熱的問題により従来使用不可であった素子の活用 ( eg. 白色LED：熱設計によって実用化 )
