=========================================================
Sphinx とは？
=========================================================

* 「ドキュメンテーションが楽しい！」という噂のアイツ．
* 実際、簡易版HTMLみたいな．
* そんなに楽しくはない．
* 多分、議事録とか、こんなんで作ってたら楽なんだろうかね．
* ReST型の文書作成言語．
* インストールは以下で解説．


  
インストール / クイックスタート
=========================================================

::

   $ sudo apt-get install python-sphinx

   
クイックスタート
=============================

Sphinx始めるときの魔法の言葉．

.. code-block:: none
   
   $ sphinx-quickstart



コンパイル
======================================

* make コマンドでいける

  + html 生成 ( $ make html   @ sphinx/labliver/ :: build と source と Makefileがみえるところ )
  + LaTeX生成 ( $ make latex )
