=========================================================
pythonとは
=========================================================

* Linux 標準のプログラミング言語．ライブラリが豊富で2022年現在、広く使用されている言語のひとつ．
* 型付けなしのスクリプト言語で、実行は遅い．一方、C/Fortranの型付け言語をライブラリとして簡単に呼べるため、うまく書けばそこそこ速い実行をさせることはできる．
* 豊富なライブラリをもつ型付けしないスクリプト型言語であることから、コーディングが非常に楽．
