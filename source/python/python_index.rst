=========================================================
pythonとは
=========================================================

* Linux 標準装備のプログラミング言語．
* 科学技術計算分野やソフトウェア作成、WEB、各種処理、と幅広い領域で使用されている言語．
* :red:`ライブラリが豊富`
* :blue:`型付けなしのスクリプト言語` で、 :red:`実行は遅い` ．
* 一方、C/Fortranの型付け言語をライブラリとして簡単に呼べるため、 :blue:`うまく書けばそこそこ速い実行をさせることはできる` ．
* 豊富なライブラリをもつ型付けしないスクリプト型言語であることから、 :red:`コーディングが非常に楽` ．
